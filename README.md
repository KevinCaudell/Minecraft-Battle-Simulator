# Minecraft-Battle-Simulator in Python
Minecraft battle simulator built using Python to demonstrate object-oriented programming (OOP) concepts and game logic.

# Features
- Player vs Mob Combat
- Multiple Battle Rounds and Difficulty
- Custom character classes with use of inheritence
- Unique abilities

# Technologies Used
- # Python #
- # Object-Oriented-Programming (OOP) #
  - Classes & Objects
  - Inheritences (Custom classes & built-in classes)
  - Encapsulation
- Randome Module
- Game logic
